{ PROCESS }

Below is an outline of my thought process as it applies to UX Design. The project showcased for this example was a proposal to design a web app that created a unified location for several resources used by a project managment team. Click on each step to learn more about how I think.

+

-

DEFINE THE PROBLEM

Employees were frustrated with keeping track of their projects on several different platforms, in several different locations. The primary issue requiring a resolution was to improve the method for task delivery. Timing and ease of task delivery would be the metric.

+

-

CONDUCT RESEARCH

Research invloved interviewing employees and observing daily routines. This process revealed employee's navigating between Box.com, two independant Microsoft Share Point interfaces, Plan Grid, Excel Files and Outlook Calendar. Employee's spent an average of 12 miniutes every hour navigating between program and file locations. In addition to these 12 miniutes, employees spent an average of 5 minuites preforming a single given task, and then another average of 5 miniutes duplicating the same task for a different program.
